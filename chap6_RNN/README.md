

# 循环神经网络(RNN)



## 问题描述：

利用循环神经网络，实现唐诗生成任务。




## 数据集: 

唐诗





## 题目要求： 

补全 `poem_generation_pytorch.ipynb` 中模型部分的两处填空：`__init__` 中 `self.rnn_lstm` 的定义，以及 `forward` 中调用 LSTM 的一行。

生成诗歌 开头词汇是 “ 日 、 红 、 山 、 夜 、 湖、 海 、 月 ”。
